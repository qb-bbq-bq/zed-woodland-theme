# Woodland

A warm, dark Zed theme — muted forest tones (bark browns, moss green) with a
pink scrollbar/cursor accent and a One Dark–style syntax palette.

Ships as the `qb-woodland` extension. Both variants are `dark`.

## Palette

| Role | Color | |
|------|-------|--|
| Background (editor) | `#17140F` | deep warm bark |
| Background (panels/bars) | `#14110D` | |
| Surface / elevated | `#14110D` / `#1E1A15` | |
| Foreground (text/icons) | `#ece0cd` | warm off-white |
| Muted text | `#a3957f` | |
| **Accent (moss green)** | `#a5bf6e` | text.accent, success, focus borders `#5C6A3E` |
| Cursor / scrollbar | `#F0AEC0` | pink — the one cool pop |
| Warning / modified | `#e0a542` | amber |
| Error / deleted | `#FF3737` / `#FF5858` | |
| Added | `#A5FF6E` | |
| Info / renamed | `#7ba0a6` | slate blue |

### Terminal ANSI
Earth-toned: red `#d97a63`, green `#a5bf6e`, yellow `#e0a542`, blue `#7ba0a6`,
magenta `#c58fae`, cyan `#8fbfa8`, white `#ece0cd` (+ bright/dim of each).

### Syntax
Same as One Dark theme

## Variants

| Variant | Description |
|---------|-------------|
| **Woodland Night** | Default |
| **Woodland Night Flat** | Flatter — transparent ghost element background (icon btns, breadcrumbs, menu items, ...) |
